A simple pong port in Unity. This is my first project, so expect bugs and a lot of weird design choices! Be kind!

### Controls

#### Paddle 1

**W** - Up!

**S** - Down!

#### Paddle 2

**UP ARROW** - Up!

**DOWN ARROW** - Down!

#### Global

**ESC** - Quit the game!
